# FlexRun - UI/UX Design Specification

**Version**: 1.0  
**Last Updated**: February 6, 2026  
**Platform**: Android (Native Kotlin)

## 1. Design System

### 1.1 Color Palette

#### Primary Colors

| Name | Light Mode | Dark Mode | Usage |
|------|------------|-----------|-------|
| Primary | `#FF6B35` | `#FF8B5E` | CTAs, active states, accent |
| Primary Dark | `#E55A25` | `#FF6B35` | Pressed states |
| Primary Light | `#FFE4D9` | `#3D2A22` | Backgrounds, containers |

#### Neutral Colors

| Name | Light Mode | Dark Mode | Usage |
|------|------------|-----------|-------|
| Background | `#FAFAFA` | `#121212` | Main background |
| Surface | `#FFFFFF` | `#1E1E1E` | Cards, sheets |
| Surface Elevated | `#FFFFFF` | `#2C2C2C` | Elevated cards |
| On Background | `#1A1A1A` | `#E5E5E5` | Primary text |
| On Background Secondary | `#666666` | `#A0A0A0` | Secondary text |
| Divider | `#E0E0E0` | `#333333` | Lines, borders |

#### Semantic Colors

| Name | Light Mode | Dark Mode | Usage |
|------|------------|-----------|-------|
| Success | `#34C759` | `#30D158` | Achievements, good pace |
| Warning | `#FF9500` | `#FF9F0A` | Pace warnings |
| Error | `#FF3B30` | `#FF453A` | Errors, stop action |
| Info | `#007AFF` | `#0A84FF` | Information |

### 1.2 Typography

**Font Family**: Inter (Google Fonts)

| Style | Size | Weight | Line Height | Usage |
|-------|------|--------|-------------|-------|
| Display Large | 57sp | Bold (700) | 64sp | Active run timer |
| Display Medium | 45sp | Bold (700) | 52sp | Stats hero numbers |
| Headline Large | 32sp | SemiBold (600) | 40sp | Screen titles |
| Headline Medium | 28sp | SemiBold (600) | 36sp | Section headers |
| Title Large | 22sp | Medium (500) | 28sp | Card titles |
| Title Medium | 16sp | Medium (500) | 24sp | List item titles |
| Body Large | 16sp | Regular (400) | 24sp | Main body text |
| Body Medium | 14sp | Regular (400) | 20sp | Secondary text |
| Label Large | 14sp | Medium (500) | 20sp | Button text |
| Label Medium | 12sp | Medium (500) | 16sp | Small labels |

### 1.3 Spacing System

| Token | Value | Usage |
|-------|-------|-------|
| `space-xs` | 4dp | Tight gaps |
| `space-sm` | 8dp | Small gaps, inline spacing |
| `space-md` | 16dp | Standard padding, gaps |
| `space-lg` | 24dp | Section spacing |
| `space-xl` | 32dp | Large section gaps |
| `space-2xl` | 48dp | Major screen divisions |

### 1.4 Border Radius

| Token | Value | Usage |
|-------|-------|-------|
| `radius-sm` | 8dp | Small elements, chips |
| `radius-md` | 12dp | Cards, buttons |
| `radius-lg` | 16dp | Bottom sheets, modals |
| `radius-full` | 9999dp | Pills, avatars |

### 1.5 Shadows

| Elevation | Shadow | Usage |
|-----------|--------|-------|
| Level 1 | `0dp 1dp 3dp rgba(0,0,0,0.12)` | Cards, chips |
| Level 2 | `0dp 4dp 6dp rgba(0,0,0,0.15)` | Floating buttons |
| Level 3 | `0dp 8dp 16dp rgba(0,0,0,0.2)` | Bottom sheets, dialogs |

## 3. Animations & Micro-Interactions

### 3.1 Screen Transitions

| Transition | Animation | Duration |
|------------|-----------|----------|
| Push (forward) | Slide from right | 300ms |
| Pop (back) | Slide to right | 250ms |
| Modal | Slide from bottom | 350ms |
| Fade | Cross-fade | 200ms |

### 3.2 Component Animations

**Button Press**:
- Scale down to 0.95
- Duration: 100ms
- Easing: EaseOut

**Quick Action Chip Selection**:
- Background color transition: 200ms
- Border appears: 150ms
- Scale up: 1.02 for 100ms

**Metric Update**:
- Count up animation for numbers
- Duration: 300ms
- Easing: EaseOutCubic

**Progress Bar**:
- Smooth width transition
- Duration: 500ms
- Easing: Linear

**Voice Coaching Message**:
- Slide up from bottom
- Fade in
- Duration: 200ms
- Auto-dismiss after 4 seconds

### 3.3 Loading States

**AI Response Loading**:
- Three dots pulsing animation
- Located in AI message area
- "Thinking..." text below dots

**GPS Acquiring**:
- Location icon pulsing
- "Finding your location..." text
- Concentric circles animation

## 4. Accessibility

### 4.1 Touch Targets

- Minimum: 48dp x 48dp
- Recommended: 56dp x 56dp for primary actions

### 4.2 Color Contrast

- Text on background: Minimum 4.5:1
- Large text: Minimum 3:1
- UI components: Minimum 3:1

### 4.3 Screen Reader Support

- All interactive elements have content descriptions
- Images have alt text
- State changes are announced

### 4.4 Font Scaling

- Support up to 200% font scaling
- Layouts adapt gracefully
- Critical info visible at all scales

---

## 5. Voice Coaching UX

### 5.1 Audio Guidelines

**Volume**: 
- Default to 80% of media volume
- Respect silent/vibrate modes unless overridden

**Ducking**:
- Lower music/podcast volume by 50% during announcement
- Resume volume 500ms after announcement ends

**Queue Management**:
- Max 1 announcement in queue
- New announcements replace queued (not current)
- Don't interrupt mid-sentence

### 5.2 Announcement Frequency

| Coaching Level | Distance | Time | Pace Alerts |
|----------------|----------|------|-------------|
| Minimal | Every 2km | None | Never |
| Normal | Every 1km | Every 5min | When off target |
| Verbose | Every 500m | Every 2min | Continuous |
